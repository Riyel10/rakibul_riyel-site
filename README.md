This repository contains everything you need to run and deploy your AI Studio application. It is designed to showcase your portfolio while integrating AI‑powered features.

📦 Features
- Modern frontend powered by Node.js
- Easy local development setup
- Integration with Gemini API
- Ready for deployment on platforms like Vercel or Render

🛠️ Prerequisites
- Node.js (latest LTS recommended)
- A valid Gemini API Key

⚙️ Installation & Setup
- Clone the repository
git clone https://github.com/your-username/personal-portfolio.git
cd personal-portfolio
- Install dependencies
npm install
- Configure environment variables
- Create a file named .env.local in the root directory.
- Add your Gemini API key:
GEMINI_API_KEY=your_api_key_here
- Run the app locally
npm run dev
- View your app
- Local: http://localhost:3000
- AI Studio: AI Studio App (ai.studio in Bing)

🌐 Deployment
You can deploy this app seamlessly on platforms like:
- Vercel: Optimized for frontend frameworks and serverless functions.
- Render: Great for full‑stack apps with persistent backend services.
Simply connect your GitHub repository to your chosen platform and enable automatic deployments.
